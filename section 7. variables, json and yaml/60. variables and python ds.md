# VARIBLES AND PYTHON DATA STRUCTURE 

## VARIABLES
Variables are temporary storage of data in the memory (RAM). So they live in the program. When we create variables in git bash, these variables will live in git bash, when we close git bash, those variables will be gone (temporary storage).

```
#create variable 
$ skill='DevOps' #string

$ NUM=123 #integer

#print variable 
$ echo $skill
$ echo $NUM
$ echo "I am learning $skill"
```
