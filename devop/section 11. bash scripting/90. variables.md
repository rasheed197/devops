# VARIABLES

Variables are basically storage in the memory.

If you logut of your computer, or you close the shell the variables will be gone.

### Create a variable
```
$ SKILL="DevOps"
```

### Print a variable
```
$ echo $SKILL
```

### Example

```
$ PACKAGE="httpd wget unzip"

$ yum install $PACKAGE -y
```
