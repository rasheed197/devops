# INTRODUCTION

### NGINX
It is a web service just like apache httpd, and it's commonly used to create a load balancing experience.

### APACHE TOMCAT 
It is a Java web application service. So if you have a web application written in java tomcat is a very famous service to host it.

### NFS SERVERS 
For external storage. If have a cluster of servers and you need an external storage, you can use NFS.

### RABBITMQ
RabbitMQ is a message broker, or also called a queuing agent.

### MEMCACHE
When a user login with username and password, our application will run an SQL query to access the information stored in MySQL database. But before it goes to MySQL database, the request will go to **memcahe service** 

Memcache is a database caching, it will be connected with MySQL server. MySQL will store the user information. The first time the request comes to the MySQL database, it will be sent from MySQL server to Tomcat and then it will be cached to the memcahe server. So next time the same request comes, it will accessing the data which is cached in the memcahe server, just like you have browser caching, we also have database caching.
