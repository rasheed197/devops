# INSTALLING SOFTWARES

- Go the github link
```
github.com/hkhcoder/vprofile-project
```
- Go to the branch "prereqs"

- All files are the same just different format

- For windows execute the command as administrator in powershell

- For macos execute the command in terminal

- See lists of installed software through chocolatey/brew
```
# Windows
choco list

# MacOS
brew list
```

- Before installing the softwares, run the following commands
```
mvn --version

# and 

java --version
```

- If you get an error then you can start installing the softwares

- But if you see some versions after running the command you need to unsinstall them using
```
choco uninstall [PACKAGE NAME]

# or 

brew uninstall [PACKAGE NAME]
```

- You can now start installing all the packages in the github file
