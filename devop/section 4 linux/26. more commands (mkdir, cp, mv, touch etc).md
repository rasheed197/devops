# MORE COMMANDS (mkdir, cp, mv, touch etc)

### 1. CREATE DIRECTORY

```
# syntax: mkdir dir1 dir2 ...
```
```
# create single directory
mkdir dev

# create multiple directory
mkdir ops backupdir textdir mobile
```

### 2. CREATE EMPTY FILE OR UPDATE FILE TIMESTAMP

```
# syntax: cp file1
```
```
# create single file
touch textfile1.txt

# create multiple file
touch devopsfile{1..10}.txt
```

### 3. COPY FILE OR DIRECTORY

```
# syntax: cp src dest
```
```
# copy a file using relative path
cp devopsfile1.txt dev/

# copy a file using absolute path
cp /home/vagrant/devopsfile2.txt /home/vagrant/dev/

# copy directory
cp -r dev backupdir/
```

### 4. MOVE OR RENAME FILES OR DIRECTORIES

```
# syntax: mv src dest
```
```
# move directories
mv ops dev/

# rename file
mv textfile1.txt textfile22.txt

# move all .txt file to textdir
mv *.txt textdir/
```

### 5. REMOVE FILES OR DIRECTORIES
```
# syntax: rm file

# remove a file
rm devopsfile10.txt

# remove a directory
rm -r mobile

# remove all files or dir in your current working dir
rm -rf *
```
