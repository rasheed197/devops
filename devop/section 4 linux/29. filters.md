## 1. SEARCH FOR TEXT IN FILE
```
# search for "firewall" in anaconda-ks.cfg
[root@centos ~]# grep firewall anaconda-ks.cfg

firewall --disabled

# ignore case sensitivity
[root@centos ~]# grep -i firewall anaconda-ks.cfg

Firewall configuration
firewall --disabled

```

## 2. IO REDIRECTION
### INDUT REDIRECTION (DEFAULT BEHAVIOUR)
"anaconda-ks.cfg" file is passed as input to the grep command (input redirection)
```
grep -i firewall anaconda-ks.cfg

# OR

grep -i firewall < anaconda-ks.cfg
```
