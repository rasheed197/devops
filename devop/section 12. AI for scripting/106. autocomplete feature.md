# AUTOCOMPLETE FEATURE

Download the extensions `Bash IDE` in VSCode.

Now if you go back to one of the script you've written and goes to the last line and hit `Enter` it would give you suggestion, press `Tab` to autocomplete. If you go to the next line again you get another suggestion like before.

This is GitHub copilot making suggestion for you based on the code you've written. So it's also learning from you.

After everything, make sure to always review your code. 

Even the AI can review your code for for you. To do that just select all the code <kbd>Ctrl</kbd>+<kbd>a</kbd>, Then click on the yellow star at the top, you will see `Review using Copilot`
