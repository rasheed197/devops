# FILE PERMISSIONS

File permissions may be viewed using
```
ls -l
```

Example
```
[aws@centos ~]$ ls -l /bin/login

-rwxr-xr-x. 1 root root 44664 Aug 22  2024 /bin/login
```

Four symbols are used when displaying permissions
```
r: permisson to read a file or list a directory content
w: permission to write to a file or create and remove file from a directory
x: permission to execute a program or change (cd) into a directory and do a long listing of the directory
-: no permission (in place of the r, w, or x)
```

```
-rwxr-xr-x. 1 root root 44664 Aug 22  2024 /bin/login

-: file type, (normal file)
rwx: user permission
r-x: group permission
r-x: others permission
root: root user owns this file
root: group root owns this file
```
