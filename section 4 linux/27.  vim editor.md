# VIM EDITOR

## INSTALL VIM EDITOR

- CentOS don't have vim editor installed by default
- We only have vi editor in CentOS
- vim is advanced vi editor
- vim is already installed in Ubuntu by default
```
# centos
sudo yum install vim -y
```

### CREATE AND EDIT A FILE WITH VIM
```
vim firstfile.txt
```

There are 3 modes in vim editor
- Command Mode
- Insert Mode (edit mode)
- extended command mode

When you open the vim editor it will be in command mode by default.

#### 1. SWITCH TO INSERT MODE
<kbd>i</kbd> or <kbd>o</kbd>

#### 2. SWITCH TO COMMAND MODE
<kbd>esc</kbd>

#### 2. SWITCH TO EXTENDED MODE
<kbd>esc</kbd> + <kbd>:</kbd>

#### 3. SAVE FILE
<kbd>esc</kbd> + <kbd>:</kbd> + <kbd>w</kbd>

#### 4. QUIT VIM
<kbd>esc</kbd> + <kbd>:</kbd> + <kbd>q</kbd>

#### 5. QUIT AND SAVE VIM
<kbd>esc</kbd> + <kbd>:</kbd> + <kbd>w</kbd> + <kbd>q</kbd>

#### 6. QUIT VIM WITHOUT SAVING
<kbd>esc</kbd> + <kbd>:</kbd> + <kbd>q</kbd> + <kbd>!</kbd>

#### 7. SHOW LINE NUMBERS
<kbd>esc</kbd> + <kbd>:</kbd> + <kbd>se nu</kbd>

#### 8. JUMP TO LAST LINE
<kbd>esc</kbd> + <kbd>Shift</kbd> + <kbd>g</kbd>

#### 9. JUMP TO FIRST LINE
<kbd>esc</kbd> + <kbd>gg</kbd>

#### 10. COPY A LINE
<kbd>esc</kbd> + <kbd>yy</kbd>

#### 11. COPY MULTIPLE LINE
Example copy 5 lines

<kbd>esc</kbd> + <kbd>5</kbd> + <kbd>yy</kbd>

#### 12. PASTE LINE BELOW
<kbd>esc</kbd> + <kbd>p</kbd>

#### 13. PASTE LINE ABOVE
<kbd>esc</kbd> + <kbd>P</kbd>

#### 14. DELETE OR CUT A LINE
<kbd>esc</kbd> + <kbd>dd</kbd>

#### 15. DELETE OR CUT MULTIPLE LINE
Delete copy 5 lines

<kbd>esc</kbd> + <kbd>5</kbd> + <kbd>dd</kbd>

#### 16. UNDO AN ACTION
<kbd>esc</kbd> + <kbd>u</kbd>

#### 17. SEARCH TEXT
Search for the word "enjoy"
press "n" to move to the next search

<kbd>esc</kbd> + <kbd>/enjoy</kbd>

#### 18. SEARCH AND REPLACE 1ST OCCURRENCE
Replace the first occurrence of "coronavirus" with "covid19" in each line

<kbd>esc</kbd> + <kbd>:%s/coronavirus/covid19</kbd>

#### 19. SEARCH AND REPLACE ALL OCCURRENCE
Replace all occurrences of "coronavirus" with "covid19" in each line (using 'g' flag for global)

<kbd>esc</kbd> + <kbd>:%s/coronavirus/covid19/g</kbd>
