# COMMAND LINE ARGUMENTS

For example, for `ls` command, we pass the path to the directory, that path is the argument.

```
$ ls path/to/dir
```

`path/to/dir` is an argument

Command line arguments is from 0 to 9

- 0 is the name of the script

- 1 - 9 are the arguments passed to the script.

This topic was implemented on `3_websetup.sh`. 

The `URL` is $1 and the `ART_NAME` is $2.

Now to run the file, we do
```
$ args_websetup.sh https://www.tooplate.com/download/2151_winter_gallery 2151_winter_gallery
```
