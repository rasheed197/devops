# THE CD COMMAND

The cd command in Linux stands for "change directory,".

It allows users to change their current working directory within the terminal.

## 1. Go to home dir from any dir
```
cd

# OR

cd ~
```
## 2. Go to the root directory
```
cd /
```

## 3. Go back one directory
```
cd ..
```

## 4. Change directory (relative path).
```
cd Downloads
```

## 5. Change directory (absolute path).
```
cd /home/rasheed/Downloads
```

## 6. Go to the previous working directory.
```
cd -
```

## 7. Go to a directory name with spaces
```
cd "My Documents" 

# OR

cd 'My Documents' 

# OR

cd My\ Documents
```
