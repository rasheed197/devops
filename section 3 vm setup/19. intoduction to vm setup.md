# VM SETUP

## INTRODUCTION
- Two popular flavor of Linux OS
    - CentOS
    - Ubuntu
- There are two method of creating VMs
    - Manual: wizard based, where we select multiple options in order to create VMs
    - Automatic: Create a text file, issue a command and the VM will be up and running

- If you want to automate something, make sure you know how to do it manually first.
- When you do a task manually, you know each and every step
- Automation is simply assembling all these steps together in a logical way by using a tool, or scripting, or any programming language

## REQUIREMENTS
Your Computer
    - 64 bit computer
    - High speed internet
    - Windows 10/11, MacOS Intel/M1 chip & Linux distros

## MANUAL STEP TOOLS
- Oracle VM VirtualBox (Hypervisor)
- ISO file (CentOS and Ubuntu)
- Login tool (GitBash and Putty)

## AUTOMATI STEPS TOOLS
- Oracle VM VirtualBox (Hypervisor)
- Vagrant (Creates VMs with Vagarantfile)
- Commands (vagrant up)
